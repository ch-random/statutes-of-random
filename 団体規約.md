# コンピュータ ハウス ランダム　団体規約  


## 第1章  総則

1. 名称　

本部活動の名称は「コンピュータ ハウス ランダム」とする。

2. 所在地　

この団体を次の住所に置く。  
大阪府堺市中区学園町1番1号　大阪公立大学文化部連合内

3. 目的　

本部活動はプログラミングについての知識の普及と、イラスト・CG・DTM制作などPCに関する創作活動に取り組み、知識・技術・想像力を発展させることを目的としている。

4. 発足　

本部活動の創立年月日は昭和59年7月7日とする。

## 第２章  活動

1. 部会

第5章で定める部会を定期的に開く。

2. Webサイトの運営

部活動に関するWebサイトを制作し、適切に運営する。

3. 大学行事

大学祭、その他の学校行事に積極的に参加する。

4. 対外活動

他団体と共同で積極的に活動し、部活動外との交流を深めるよう努力する。  
また、プログラミングコンテストに積極的に参加する。

5. その他 

本部活動の目的達成の為、その他の必要な活動を積極的に行う。  

## 第3章  部員

1. 構成　

本部活動の部員は、大阪公立大学生および、同院生で構成される。

2. 現役部員

学部・学域生1、2、3回生から構成される。現役部員の義務および権利は3.3、3.4に定められる。

3. 現役部員の義務  
1. 作品制作

現役部員はPCを用いた創作活動に取り組み、活動を報告・公開しなければならない。

2. プログラミング

現役部員はプログラミングの勉強会に出席し、基礎的な知識を身に着ける。

3. 部会の出席

現役部員は原則として部会に出席しなければならない。

4. 部費の納入

現役部員は一定期間ごとに部会で承認された部費を納入しなければならない。

5. その他

現役部員は本部活動の目的達成の為、その他の必要な活動を行わなければならない。

4. 現役部員の権利  
1. 部室

現役部員は部室の鍵を貸与され、備品を利用することができる。

2. 備品

現役部員は部会の承認を経て、備品を借りることができ、活動に必要な備品を部費から購入することができる。

3. Web

現役部員はWebサイトのログイン権を持ち、一定の権限が与えられる。

5. 特別部員

学部・学域生4回生、大学院生から構成される。現役部員の承認を得て決定され、引退後も部活動に関与することができる。特別部員の義務および権利は3.6、3.7に定められる。

6. 特別部員の義務  
1. 現役部員とのコミュニケーション

特別部員は現役部員への交流・指導に積極的に取り組む。また、部活動の規約・方針に従い、現役部員からの連絡の確認は怠らないようにしなければならない。

2. その他

特別部員は本部活動の目的達成の為、その他の必要な活動を行わなければならない。

7. 特別部員の権利  
1. 部室

特別部員は部室の鍵を貸与され、備品を利用することができる。

2. OB会

特別部員はOB会に参加することができる。

## 第4章  入退部・引退

1. 入部

本部活動への入部は、部が定める形式に沿って行われる。部長が承認することで、現役部員としての所属・活動が認められる。入部期間は設けておらず、申請はいつでも認められる。

2. 退部

何らかの理由により本部活動の継続が困難になった場合、部が定める形式に沿って退部が認められる。退部者は未払いの部費の納入や鍵の返却を行わなければならない。退部後、現役部員の義務及び権利は失効され、一年間は再入部できない。

3. 引退

3回生の現役部員は1、2回生の現役部員の承認がなければ、3回生終了時に引退となる。引退後、現役部員の義務及び権利は失効される。

## 第5章  部会

1. 部会

本部活動は必要に応じて部会を開く。部会では活動に対する報告や議論、5.2に定める投票・承認を行う。書記は部会議事録を記録する。

2. 投票・承認

部活動や部員に対して何らかの承認が求められる場合、部会で投票を行う。投票権は出席した現役部員のみが持ち、過半数の賛成を得ることで承認される。

3. 委任状

やむを得ない理由で部会を欠席する場合、定められた形式に基づいて委任状を提出しなければならない。委任状を提出した部員の投票権は無効とする。

## 第6章  役員会

1. 役員会

創作活動以外の事務報告や活動方針の共有・相談を円滑に行い、部活動を維持するために役員会を設ける。部長・副部長・会計・副会計・書記・庶務・Webの7つの役職を原則として設け、必要に応じて役職を設立する。ただし、部長は1名とし、本部活動の代表者となる。役員は現役部員から選出しなければならない。

2. 選出

役員は原則、前任役員もしくは部長が推薦し、同意を得ることで選出される。選出された役員は部会での承認によって正式に決定される。

3. 任期・交代

役員会の任期は任命時期から原則1年間とする。役員の交代は11月下旬から12月上旬に行い、この期間中に前任役員が後任役員に対して引継ぎを行う。

4. 辞任

役員がやむを得ない理由で役割が果たせなくなったとき、役員会に申し出なければならない。辞任は後継者へ引継ぎした上で、役員会の過半数の同意によって承認される。 

## 第7章  部費・会計関連

1. 部費

部費は部活動の維持・発展のために用いられる。会計は部員から部費を徴収し、責任をもってこれを管理する。また、通信料をはじめとした部活動にかかわる各種の契約は会計が責任を負うものとする。

2. 鍵の費用

部室の鍵のレンタル代金として、入部時に会計が定めた金額を渡さなければならない。この代金は鍵を返す際に返却される。

3. 会計年度

本部活動の会計年度は毎年4月1日に始まり、翌年3月31日に終了するものとする。

4. 会計報告

会計は原則年1回、会計報告書を作成する。必要に応じて部会や役員会で共有する。

## 第8章  警告・除名

1. 警告

以下に該当する現役部員に対しては役員会より警告を発する。

* 部会を3回以上無断で欠席する   
* 部費を滞納する   
* 本部活動の目的に対する活動を著しく怠る  
    
2. 除名

以下に該当する現役部員に対しては、役員会の承認を得て除名することができる。

* 複数回の警告に対して、態度が改められない  
* 本部活動や他の部員の名誉を著しく阻害する

また、以下に該当する特別部員に対しても、役員会の承認を得て除名することができる。

* 特別部員の義務を守らず、現役部員の利益を損なう  
* 本部活動や他の部員の名誉を著しく阻害する

除名された部員は指定された形式に従って、未払いの部費の支払いや鍵の返却を直ちに行わなければならない。  

## 第9章  改定

1. 改定

本団体規約は部会において現役部員の過半数の賛成により改定される。
